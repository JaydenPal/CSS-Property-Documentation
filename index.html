Font Properties

property values description
font-size ___px changes size of font
font-weight bold changes thickness of font
font-family fontName changes font of typeface

Element Appearence Properties

property values description
color colorVal sets color element appears in
background-color colorVal sets color background of element appears in
opacity val 0-1 sets transparency of element; 0 transparent; q opaque
background-image url("file/path") sets background of element to image
